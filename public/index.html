Nostr Indexer API

This project exposes serverless endpoints and a scheduled indexer.

  • /api/health
  • /api/indexer-stats
  • /api/indexer-cron (invoked by Vercel Cron)
  • /api/search?q=...
  • /api/profile/<pubkey-or-npub>
  • /api/following/<pubkey>
  • /api/followers/<pubkey>
  • /api/stats/<pubkey>
